Tentang Saya

Nama saya Muhamad Arief Rahmatulloh Saya adalah seorang Mahasiswa di STT Mandala Bandung jurusan S1 Teknik Informatika. Dan saya saat ini sedang kuliah di semester ke 2, juga sedang menjalani Ujian Tengah Semester. Saya baru berusia 19 tahun tepatnya 17 Februari 2021 kemarin usia saya sudah mencapai 19 tahun. Saya tinggal bersama orangtua saya di Rancaekek, Kabupaten Bandung.

Saya mempunyai hobi yaitu ngoding web dan juga Desain Grafis, terkadang saya juga suka menggambar animasi khususnya animasi Jepang seperti Naruto atau Dragon Ball. Kadang juga saya suka menonton film terlebih film aksi dan petualangan. Saya juga sering nonton review-review tentang teknologi di Youtube sekaligus nonton tutoria-tutorial mengenai bahasa pemrograman.
